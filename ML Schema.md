## Vertical interoperability
- Interactions between elements at different levels in an architecture is possible
- In this case, **working with ML platforms that have different data representations**

## Horizontal scaling
- Interactions between elements at the same level in an architecture is possible
- In this case, **interoperability among state-of-the-art (SOTA) ML ontologies and vocabularies**.
