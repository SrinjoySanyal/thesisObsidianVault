## Variables
Bound to **RDF terms**
## `SELECT`
- Used to **query for variables**
- Query results returned as **table**
## Graph pattern (Triple pattern)
- Contains variables at arbitrary places
- **Turtle + Variables**
- SPARQL is based on
	1. **RDF Turtle serialisation**
	2. **Basic graph pattern matching**
- SPARQL graph patterns can be combined to form **complex (conjunctive) queries** that are used to traverse RDF graphs
